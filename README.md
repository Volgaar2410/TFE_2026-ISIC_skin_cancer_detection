# TFE 2026 - Wei Zhou

Code associated with the Master thesis (TFE 2026) on skin cancer detection using the ISIC 2024 dataset.

This work is based on the following Kaggle baseline notebook:
https://www.kaggle.com/code/motono0223/isic-pytorch-training-baseline-image-only

The baseline is extended with additional experiments, including architectural modifications, attention mechanisms, self-supervised pretraining, and multimodal integration.

Dataset is not included in this repository.
